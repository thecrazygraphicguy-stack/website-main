# The Crazy Graphic Guy — Website

Static website for thecrazygraphicguy.com

## Stack
- HTML/CSS/JS (no build step)
- Fonts: TASA Orbiter, DM Sans, Orbitron (Google Fonts)
- Deploy: Cloudflare Pages

## Pages
- `/` — Homepage
- `/products/` — YouTube CTAs category
- `/products/cta-001/` through `/products/cta-009/` — Product pages
- `/about/` — Who is TCGG
- `/contact/` — Contact & Help
- `/wall-of-love/` — Wall of Creators
- `/terms-of-service/` — Terms
- `/refund-policy/` — Refund policy

## Deploy
Push to GitHub → connected to Cloudflare Pages → auto-deploys.

## Local dev
Open any `index.html` directly in browser, or use:
```
npx serve .
```
